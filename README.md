# Keycloak Invite Code Authenticator

![A screenshot of the authentication flow asking for an invite code.](.github/images/invite-code-screenshot.jpg)

An authenticator for [Keycloak](https://keycloak.org) to require an "invite code" to proceed during an authentication flow. Namely adding it as a step for user registrations.

> [!WARNING]
> This was quickly thrown together and my experience with Java is **very** minimal.
